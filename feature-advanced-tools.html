Programs Grow. Rules Multiply.

What starts as a handful of promotions becomes hundreds of rules. Base earn. Partner earn. Tier multipliers. Promotional bonuses. Each rule interacts with the others. Without structure, you get unknown outcomes. Which rules fired? In what order? Why did the member get 150 points instead of 250? That's a liability problem. An audit problem. A finance problem.

Rule Organization

Predictable Execution. Auditable Outcomes.

FlowComposer organizes your rules into a hierarchy that matches how your organization actually works. Base earn in one folder. Partner rules isolated per partner. Tier logic separate from promotions. Different teams own different folders with role-based access. When a transaction comes through, the execution path is explicit: Load Data, Base Rules, Promotions, Partner Rules, Tier Rules, Exit. Every decision is logged. Finance can trace outcomes to rules. Partner billing calculates accurately because execution order is defined, not discovered.

  • Hierarchical folder structure for rule organization
  • Role-based access control at folder level
  • Folder gate rules that control activation of entire sections
  • Explicit execution order eliminates unknown outputs
  • Finance-grade logging for audit and reconciliation

Custom Logic

Build What Doesn't Exist Out of the Box

When standard configuration reaches its limits, Expression Builder provides full programmability. This is a fully programmable engine in JavaScript. You can create any logic you need. Reference any attribute in the system: activity data, member profile, preferences, transaction history. Create calculated fields that can be reused across rules. Validate expressions against real test data before they go live. Other platforms say "we don't do that." We say "here's how."

  • Full JavaScript programmability for custom logic
  • Reference any attribute: activity, member, preferences, history
  • Create reusable calculated fields (aliases) for complex qualifications
  • Validate expressions against test data before deployment
  • Immediate availability across APIs, UIs, and data warehouse

Enhance the Experience with AI

AI Assist works directly inside Expression Builder. Describe what you want in plain English and get working code. Highlight a section and ask for improvements. The AI understands your program's existing expressions and available functions, so suggestions fit your context.

Governance at Scale

Built for Teams. Built for Audit.

The tools that keep complex programs manageable over time.

Different teams. Different folders.

Partner teams manage partner folders. Tier teams manage tier logic. Each team sees only what they need. Changes stay isolated. Collisions stay rare.

Every decision. Fully traced.

Transaction logs show which rules triggered, which actions executed, and what data informed the decision. Disputes resolve in minutes, not hours.

Each partner as a mini-program.

Separate funding, billing, and reimbursement logic per partner. Agreements stay isolated. Finance sees clean settlement with full traceability.

Structure that lasts.

Programs that run for years stay organized. New promotions drop into the right folder. Rules don't degrade into spaghetti. The system scales with the business.
