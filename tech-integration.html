Connectors for the Modern Stack

Pre-built integrations that accelerate time-to-value. Custom connectors when you need them.

Point of Sale

Oracle MICROS, NCR, Toshiba, Toast, Square, Shopify POS. Real-time transaction capture and discount application at checkout.

CRM Systems

Salesforce, HubSpot, Microsoft Dynamics. Bi-directional sync of member profiles, activity, and loyalty status.

Customer Data Platforms

Segment, mParticle, Tealium, Adobe Experience Platform. Enrich loyalty data with behavioral signals from across the customer journey.

Marketing Automation

Salesforce Marketing Cloud, Braze, Klaviyo, Iterable. Trigger journeys and inject dynamic loyalty content into campaigns.

eCommerce Platforms

Shopify, Magento, BigCommerce, Salesforce Commerce Cloud. Loyalty at checkout, account pages, and order confirmation.

Mobile Apps

iOS and Android SDKs. Push notifications, in-app messaging, and real-time balance updates. Native and React Native support.

Batch Processing

FeedXChange: Batch Files, Real-Time Processing

Not every partner sends data in real time. FeedXChange accepts batch files—CSV, XML, JSON, EDI—and processes each record through the same real-time rules engine. Every transaction evaluated. Every event triggered. Same outcomes regardless of how data arrives. Partners onboard in 4-5 hours on average.

  • Multiple file formats supported
  • Configurable field mapping
  • Error handling and reconciliation
  • Finance-grade audit trails

Publish Events the Moment They Happen
